# Hospitality Concept Visualizer Handoff Package

This folder contains the minimal Streamlit demo and supporting notes for the High Level Design report.

## Included Files

- `app.py` - Main Streamlit application
- `requirements.txt` - Python dependencies
- `.env.example` - Example environment variable names
- `PROJECT_OVERVIEW.md` - Quick summary of the project, tech stack, and structure
- `HLD_REPORT_NOTES.md` - Section-wise notes mapped to the HLD report table of contents

## Demo Summary

This application demonstrates multimodal generative AI for hospitality concept visualization.

- Input: hospitality prompt text
- Output 1: concept description generated by Google Gemini
- Output 2: concept image generated by Hugging Face Stable Diffusion

## Tech Stack

- Python
- Streamlit
- Google Gemini API via `google-genai`
- Hugging Face Inference API
- `requests`

## Run Instructions

Open PowerShell in this folder and run:

```powershell
python -m venv venv
venv\Scripts\activate.bat
pip install -r requirements.txt
$env:GEMINI_API_KEY="your_gemini_api_key"
$env:HUGGINGFACE_API_KEY="your_huggingface_api_key"
venv\Scripts\python.exe -m streamlit run app.py --server.address 127.0.0.1 --server.port 8501
```

Open the app in a browser:

```text
http://127.0.0.1:8501
```
